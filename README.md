## Run Locally
Clone the Repository

Repository Link

```bash
  git clone git@github.com:adityagarg2198/manufac_task.git
```

Go to the project directory

```bash
  cd manufac_task
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```


## Production

To generate production build this project run

```bash
  yarn run build
```
